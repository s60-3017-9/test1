- สร้างmodel ชื่อ"number"
  - number_text เก็บค่าของตัวเลขที่ใส่ใน input เพื่อให้แสดงผลการคูณ
  - count จำนวนครั้งที่เคยใส่ตัวเลขนั้นไปใน input
- สร้างเงื่อนไขในฟังก์ชันที่ใช้รับค่า input
  - if เมื่อเป็นการใส่เลขนั้นใน input ครั้งแรก จะสร้างทำการสร้าง object ของตัวเลขนั้น
    -  number_text = เลขที่ใส่ใน input
    -  count = จำนวนครั้งที่ใส่ตัวเลขนั้น(เริ่มต้นที่ 1)
  - else เมื่อเคยใส่ตัวเลขนั้นไปแล้ว
    -  count+=1 
- สร้าง template ที่ใช้แสดงค่าของจำนวนครั้งที่ใส่เลขนั้นใน input
- สร้าง ตารางที่ใช้แสดงค่าใน template
- แก้ไข model โดยเพิ่ม number_int เก็บค่าตัวเลขที่ใส่เป็น int เพื่อใช้เรียงลำดับค่า
